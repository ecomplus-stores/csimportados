# Descrição do produto

Descrição:
O Conversor VGA para HDMI 1080p é projetado para transmitir sinal de vídeo analógico para um dispositivo com entrada digital HDMI com formato máximo de até 1080p. Além de vídeo, o aparelho também pode converter o sinal de áudio de entrada para os formatos digitais, em seguida, a partir da saída para HDMI, reproduzindo junto com o vídeo.
Compatível com PC, Notebook, TVs em geral.

- Suporta HDMI 1080p
- Resolução de Saída HDMI: 720p / 1080i / 1080p
- Conexão de áudio P2 de 3,5mm
- Alimentação: 5Vdc (via porta USB)
- Dimensões: 60mm x 55mm x 22mm

#Pode não ser compatível com alguns modelos de DVR
